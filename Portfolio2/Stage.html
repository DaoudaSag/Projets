Stage

Durant mon BTS, j'ai eu l'occasion d'effectuer un stage d'une durée de 10 semaines au sein d'une entreprise de growth hacking. Le growth hacking consiste à élaborer et déployer des techniques, appelées hacks, dans le but de démultiplier la croissance d’une organisation, il est donc judicieux de cibler principalement les start-up.
La technique utilisée par mon entreprise d'accueil était l'utilisation de robots opérant sur la version web des réseaux sociaux tels que Facebook, Twitter ou encore Instagram, ayant pour but de simuler des comportements humains en menant des actions telle que liker des commentaires, s'abonner à des comptes ou encore poster des messages, le tout sous l'identité d'un vrai compte utilisateur identifié au préalable sur la plateforme.
Ces robots étaient développés sous le langage informatique Python et de son framework (extension du programme lui permettant de plus grandes fonctionnalités) Python Selenium permettant le web scrapping (technique d'extraction d'un élément sur un site Web dans le but de l'exploiter pour permettre son utilisation dans un autre contexte). C'est donc ici que j'interviens : dans la première partie de mon stage, mon rôle consistait à tester ces scripts puis à corriger les éventuels bugs qu'il pourrait y avoir afin de rendre le programme à nouveau fonctionnel ou de l'optimiser. Dans une seconde partie, ma mission était de développer mes propres scripts capables d'extraire certaines données sur des annuaires de start-up afin de les renseigner dans une base de données, créée par mes soins également.

Mes missions

 I - Première partie du stage

 - Apprendre le framework Python Selenium
 - Tester puis débugger les scripts


 II - Deuxième partie du stage

 - Web scrapping
 - Développement de scripts
 - Conception et enrichissement d'une base de données
